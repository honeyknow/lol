# Frontend Source

Source for the React dashboard.

| Folder/file | Purpose |
|---|---|
| `main.tsx` | React entrypoint. |
| `App.tsx` | View routing and top-level layout. |
| `api/` | Axios client and TypeScript API types. |
| `pages/` | Overview, Threat Hunt, Firehose, Rules Engine pages. |
| `components/` | Process tree, evidence drawer, alert queue, topbar, and related UI. |
| `assets/` | Image/static assets used by the dashboard. |
